输出代码改动时严格按本模板组织。占位符用尖括号 `<...>` 表示，代码用代码块包裹，**给出可直接复制的完整代码**。

## 新建一个类

> 在 `<相对路径/包名>` 下新建一个名为 `<类名>` 的类。这个类包含的成员变量有：

```java
private aaa bbb;
private final ccc ddd;
private final static eee fff;
```

> 其中：
> 1. `bbb` 用于 `<说明用途>`；
> 2. `ddd` 用于 `<说明用途>`；

> [如果包含]成员方法有：
> 1. `<完整方法签名，例：public aaa bbb(ccc ddd)>`

```java
public aaa bbb(ccc ddd) {
    // 方法体
}
```

> 这个方法用于 `<作用>`。它返回 `<返回值类型>`，用于 `<返回值用途>`。
> 其中关键逻辑：
> a. `<代码段>` 的逻辑是：`<阐述>`；
> b. `<代码段>` 的逻辑是：`<阐述>`；

> （所有方法说明完后）完整代码如下：

```java

```
